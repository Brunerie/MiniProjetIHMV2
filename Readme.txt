Tout d'abord, pour commencer à utiliser l'application appuyée sur le bouton "Start".

Vous y trouverez alors 4 boutons :
- Un bouton "Nouveau" vous permettant de créer de nouveaux contact et lorsque vous remplirez le champ "email" et "téléphone", veillez à vérifier que ces formats sont valides 
sinon un message d'erreur apparaîtra et vous empêchera de créer un nouveau contact. Le contact s'affichera sur la page d'accueil une fois que vous l'aurez créé.

- Un bouton "Editer" vous permettant de modifier les informations d'un contact lorsque vous le sélectionnez.

- Un bouton "Supprimer" vous permettant de supprimer un contact, et ses informations, lorsque vous le sélectionnez.

- Et un bouton "Rafraichir" vous permettant de rafraîchir la page si un contact n'apparaît pas directement.

Enfin vous y trouverai aussi des raccourcis vous facilitant l'utilisation de l'application lorsque vous passez votre souris sur un bouton.
